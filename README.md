## Classy

### O que é?

O desenvolvimento do Classy tem como objetivo servir como um sistema para a prática e o aprofundamento dos meus estudos em desenvolvimento de software. Por meio desse projeto, é possível aplicar na prática conhecimentos como modelagem de sistemas, organização de código e implementação de funcionalidades.

Apesar de ser um projeto voltado ao aprendizado, o Classy foi idealizado como um sistema escolar completo e abrangente. A proposta é simular um ambiente educacional real, com funcionalidades relacionadas ao gerenciamento de alunos, professores, turmas e disciplinas. Dessa forma, o sistema funciona tanto como ferramenta de estudo quanto como um modelo de sistema acadêmico.
